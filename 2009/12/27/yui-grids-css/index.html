YUI Grids CSS

相对挨踢

从前做页面都是自己一行一行写CSS，我不是界面方面的高手，所以通常都会让我很痛苦。

最近的项目决定使用现成的框架试试，暂时选中了YUI Grids CSS，感觉器设计是相当的优雅清晰，更酷的是，YUI Grids Builder能方便地生成你想要的布局，从此世界美好了。

记录几点：

1、计算：非IE/13，IE/13.333

2、宽度：doc 750px, docs 950px, doc3 100%, doc4 974px, doc-custome-自定义

3、任何界面都可以划分为main block和secondary block

4、模板：yui-t1 160px, yui-t2 180px, yui-t3 300px, yui-t4 180px右, yui-t5 240px右, yui-t6 300px右

5、内嵌布局：yui-u, yui-g均分, yui-gc 2/3, yui-gd 1/3, yui-ge 3/4, yui-gf 1/4, yui-gb 1/3x3 首格.first

6、链接：http://developer.yahoo.com/yui/grids/
